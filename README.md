## Sobre o Projeto

A **DevAgency** é uma plataforma web inovadora, projetada para otimizar o processo de recrutamento e gestão de desenvolvedores Fullstack Pleno e Sênior. Nossa missão é criar uma ponte eficiente e confiável entre talentos altamente qualificados e empresas que buscam equipes ágeis e de alta performance. Seja você um desenvolvedor buscando sua próxima oportunidade ou uma empresa em busca do profissional ideal, a DevAgency simplifica essa conexão.

## Funcionalidades Principais

*   **Cadastro Detalhado de Desenvolvedores:** Perfis técnicos completos, destacando habilidades, experiência e portfólio.
*   **Dashboard Administrativo Intuitivo:** Ferramentas robustas para avaliação, filtragem e gerenciamento de candidatos pelas empresas.
*   **Gestão de Vagas Simplificada:** Empresas podem publicar e gerenciar suas oportunidades de forma fácil e eficaz.
*   **Monitoramento de Status e Performance:** Acompanhamento do progresso dos candidatos e scores técnicos para uma tomada de decisão embasada.

## Tecnologias Utilizadas

Este projeto foi construído com as seguintes tecnologias modernas:

*   **Frontend:**
    *   **React:** Biblioteca JavaScript para construção de interfaces de usuário reativas.
    *   **TypeScript:** Superset do JavaScript que adiciona tipagem estática para maior robustez.
    *   **TailwindCSS:** Framework CSS utilitário para um desenvolvimento rápido e customizável.
*   **Backend & Banco de Dados:**
    *   **Firebase:** Plataforma de desenvolvimento de aplicativos do Google, utilizada para:
        *   **Authentication:** Gerenciamento seguro de usuários.
        *   **Firestore:** Banco de dados NoSQL flexível e escalável em tempo real.
*   **Deploy:**
    *   **Vercel / Firebase Hosting:** Plataformas de hospedagem e deploy contínuo.

## Instalação e Execução (Para Desenvolvedores)

Para configurar e rodar o projeto localmente, siga os passos abaixo:

1.  **Clone o Repositório:**
    ```bash
    git clone https://github.com/seuusuario/devagency.git
    ```
2.  **Acesse o Diretório do Projeto:**
    ```bash
    cd devagency
    ```
3.  **Instale as Dependências:**
    ```bash
    npm install
    ```
4.  **Configure as Variáveis de Ambiente:**
    Crie um arquivo `.env` na raiz do projeto e configure as chaves de API do Firebase. Um exemplo pode ser encontrado em `.env.example`.
5.  **Inicie o Servidor de Desenvolvimento:**
    ```bash
    npm run dev
    ```
    O aplicativo estará disponível em `http://localhost:3000` (ou outra porta configurada).
